Course Project Documentation

Summary

The purpose of this website is to create a community-driven platform for video game enthusiasts to share, discover, and discuss their favorite video games. The site aims to provide an engaging experience by offering users the ability to submit their favorite games, interact with other users through comments and discussions, and explore a curated list of top-rated games.

Website Structure / Site Map

The website is structured as follows:

  • Homepage: The homepage serves as the central hub where users can discover top-rated games, submit their favorite games, and navigate to other sections of the site.
  • Submit Your Favorite Game: This section provides users with a form to submit details about their favorite game. Upon submission, the game is added to the list of all favorite games.
  • All Favorite Games: Here, users can explore a comprehensive list of all submitted favorite games. The list is initially populated with three default games but dynamically updates as users submit their own favorites.

Button Descriptions and User Actions

Users can interact with the site through the following actions:

  • View All Favorites: This button redirects users to the "All Favorite Games" page, where they can explore the complete list of submitted favorite games.
  • Submit Your Favorite Game: Clicking this button navigates users to the "Submit Your Favorite Game" form, allowing them to share details about their favorite game with the community.
  • Menu: The top left of the webpage provides a menu button for easy navigation. Once the menu button is clicked, the user will see buttons to navigate to the previously mentioned "View All Favorites" and "Submit Your Favorite Game" buttons found on the home page.
